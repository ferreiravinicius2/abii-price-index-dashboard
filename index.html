How to read this dashboard
Corona vs M1 (local leader) and vs Heineken - always same container (bottle vs bottle, can vs can).
PI = Corona ÷ Competitor × 100
PI 107 = Corona +7%  |  PI 95 = Corona −5%  |  100 = parity

Calculation order (per retailer)

Rule 1 · 6pk Corona 6-pack vs M1 6-pack (+ HNK 6pk if present). STAR 6PK when Corona & M1 both exist at 6pk.
Rule 2 · Single No 6pk: match single-unit bottle or can with same volume.
Rule 3 · Other Exhausts every remaining Corona pack present at the retailer (12, 24, 4, 8, 18, 3, 10, 15, 2, 5, 20, 30, etc.) until a same-volume, same-container match is found.
Rule 4 · Weighted No direct match: Corona PPHL is weighted by internal Vol KHL from others/Weight.xlsb; M1 and HNK remain simple averages.
Rule 4 · Country agg. Country-level cross-retailer aggregation — Corona from any retailer vs M1/HNK from any retailer, for a national reference.

Guard-rails (Corona PI vs band)

OK PI inside the Corona band for the country tier (e.g. Corona/M1 150-180 in Developed).
OK* PI outside the band by ≤ 10 PI points (slightly off-target).
NOK PI outside the band by > 10 PI points. Action required.
NA No PI available (no Corona, no M1 / HNK, or no band for the country).

Shows the exact SKUs used in each country's match computation (rule 1/2/3 only). Rule 4 aggregates all packs.

Rule reference

  • Rule 1 - 6pk Corona 6-pack vs M1 6-pack (HNK 6pk too if present). Always same container (bottle vs bottle, can vs can). STAR 6PK MATCH applies when Corona + M1 both at 6pk.
  • Rule 2 - Single If no 6pk: fallback to single-unit match (bottle or can) with same volume.
  • Rule 3 - Other Iterates 12 → 24 → 4 → 8 → 18 → any other pack until a same-container match is found.
  • Rule 4 - Weighted No pack match per retailer: Corona PPHL weighted by internal Vol KHL (others/Weight.xlsb); M1 and HNK stay as simple averages.
  • Rule 4 - Country agg. Country-level cross-retailer aggregation: Corona (any retailer, weighted) vs M1 and HNK (any retailer, simple avg). Useful when Corona and M1 live in different retailers.
